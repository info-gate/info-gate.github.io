OpenDocs 개인정보처리방침

🔒 100% 로컬 처리 — 파일은 기기 밖으로 나가지 않습니다
시행일: 2026-04-26 · 최종 업데이트: 2026-04-26

주식회사 인포게이트(Info-Gate Inc., 이하 "회사")는 OpenDocs 앱(이하 "앱")의 개인정보 보호를 매우 중요하게 생각합니다. 본 방침은 앱이 어떤 정보를 어떻게 처리하는지 투명하게 안내합니다.

1. 수집하는 개인정보 항목

OpenDocs 는 사용자의 개인 식별 정보(이름, 이메일, 전화번호, 계정 등)를 일절 수집하지 않습니다. 회원가입·로그인 기능이 없습니다.

로컬 처리 데이터 (기기에만 저장)

  • 최근 열어본 파일 목록 (파일명, 경로, 마지막 열람 시각, 파일 크기)
  • 즐겨찾기 표시한 파일 목록
  • 앱 설정 (언어, 뷰어 옵션 등)

위 데이터는 기기 내 SQLite DB(opendocs.db)에만 저장되며 외부로 전송되지 않습니다. 앱 삭제 시 함께 제거됩니다.

광고 관련 (Google AdMob)

  • 광고 ID (Android) / IDFA (iOS): Google 의 광고 게재용 비식별 식별자
  • 기기 정보: 모델, OS 버전, 언어 (광고 타겟팅용 비식별)
  • 광고 노출/클릭 통계: Google AdMob 이 자동 수집

이 데이터의 처리·보관은 Google 의 개인정보처리방침에 따릅니다. 사용자는 기기 설정에서 광고 ID 재설정 또는 맞춤형 광고 거부 가능합니다.

분석 (PostHog) · 에러 트래킹 (Sentry)

  • 익명 사용 통계: 화면 방문, 기능 사용 빈도 (개인 식별 정보 미포함)
  • 앱 오류 로그: 크래시 발생 시 stack trace, 기기 정보

위 데이터는 앱 개선 목적으로만 사용되며, 개인을 식별할 수 없는 형태로 처리됩니다.

2. 파일 처리 방침 — 핵심 약속

사용자가 OpenDocs 로 여는 모든 문서는 기기 내에서만 처리됩니다. 파일 내용은 어떤 외부 서버로도 전송되지 않습니다. 클라우드 동기화·백업 기능을 제공하지 않으며, 파일은 사용자 기기 외부에 저장되지 않습니다.

3. 개인정보 보유 및 이용 기간

  • 로컬 처리 데이터: 사용자가 앱 내에서 삭제하거나 앱을 제거할 때까지
  • 광고 데이터 (Google AdMob): Google 정책에 따름 (광고 ID 재설정 시 즉시 변경)
  • 분석/에러 로그 (PostHog/Sentry): 90일 후 자동 파기

4. 개인정보 제3자 제공

회사는 사용자의 개인정보를 외부에 제공하지 않습니다. 다음 외부 서비스가 비식별 데이터를 자체 처리합니다 (위탁 처리 위탁):

  • Google AdMob (Google LLC, USA): 광고 게재
  • PostHog (PostHog Inc., USA): 익명 사용 통계
  • Sentry (Functional Software Inc., USA): 앱 오류 모니터링

5. 사용자 권리

  • 열람·삭제: 앱 설정 → 데이터 관리 → 캐시 삭제 또는 앱 자체 삭제로 모든 로컬 데이터 즉시 제거
  • 광고 차단: 기기 설정 → 개인정보 → 광고 → 맞춤형 광고 사용 안 함
  • 분석 거부: 앱 설정 → (Phase 2 도입 예정) 분석 옵트아웃

OpenDocs Privacy Policy (English Summary)

OpenDocs is a free document viewer that prioritizes your privacy. Key points:

  • No personal data collected: No accounts, emails, or identifying information.
  • Files stay on your device: All documents are processed locally. Nothing is uploaded to any server.
  • Local data only: Recent file list and favorites are stored in your device's local database, removed when you delete the app.
  • Advertising: Google AdMob serves ads using non-identifying device data (ad ID, OS version). Subject to Google's Privacy Policy.
  • Analytics: PostHog (anonymous usage stats) + Sentry (crash reports). No personally identifiable information.
  • Your rights: Clear all local data via Settings → Data Management → Clear Cache, or simply delete the app.
