OpenDocs 계정/데이터 삭제 안내

🔒 OpenDocs 는 계정 시스템이 없습니다 — 모든 데이터는 기기 내 로컬 저장

핵심 안내

OpenDocs 는 회원가입·로그인 기능이 없는 앱입니다. 회사의 외부 서버에 사용자 정보가 저장되지 않으며, 모든 데이터는 사용자 기기 내에서만 관리됩니다.

로컬 데이터 삭제 방법

다음 두 가지 방법으로 OpenDocs 가 기기에 저장한 모든 데이터를 즉시 삭제할 수 있습니다:

방법 1 — 앱 내 캐시 삭제 (부분 삭제)
앱 실행 → ⚙ 설정 탭 → "📂 데이터 관리" → "캐시 삭제" → 확인
※ 임시 파일만 삭제됩니다. 최근 파일 목록·즐겨찾기는 유지.
방법 2 — 앱 삭제 (전체 데이터 제거)
iOS: 홈 화면에서 OpenDocs 아이콘 길게 누름 → "App 제거" → "App 삭제"
Android: 설정 → 앱 → OpenDocs → 제거
※ 앱 삭제 시 OpenDocs 가 저장한 모든 데이터(최근 파일 목록, 즐겨찾기, 캐시)가 즉시 영구 삭제됩니다.

삭제되는 데이터 목록

  • 최근 열어본 파일 목록 (파일명, 경로, 마지막 열람 시각, 크기)
  • 즐겨찾기 표시 목록
  • 앱 설정 (언어, 뷰어 옵션)
  • 임시 캐시 파일

삭제되지 않는 것

  • 실제 문서 파일: 사용자가 다른 위치(파일 앱, 다운로드 폴더 등)에 저장한 원본 파일은 OpenDocs 와 무관하게 그대로 유지됩니다.
  • 광고 ID: 기기 시스템 ID 로 별도 관리됩니다. 재설정은 기기 설정 → 개인정보 → 광고 → "광고 ID 재설정" 에서 가능합니다.

외부 서비스 데이터 처리

OpenDocs 가 사용하는 외부 서비스는 비식별 데이터를 자체 정책에 따라 처리합니다:

  • Google AdMob: Google 개인정보처리방침 · 광고 ID 재설정 시 즉시 변경
  • PostHog (분석): 90일 후 자동 파기 · 개인 식별 정보 미수집
  • Sentry (오류 로그): 90일 후 자동 파기

Account/Data Deletion (English Summary)

OpenDocs has no account system. All data is stored locally on your device. There is no server-side data to delete.

  • Method 1 — In-app clear: App → Settings tab → Data Management → Clear Cache
  • Method 2 — Delete app: Removes all OpenDocs data (recent files, favorites, cache) instantly. Original document files in your file system are unaffected.
  • External services (AdMob/PostHog/Sentry) handle non-identifying data per their own policies.
